# Frontend

This folder contains the Next.js user interface for Varaksha demonstrations.

## Purpose

- Visualize live verdict streams and risk behavior.
- Present architecture and timeline pages.
- Exercise API integration paths for transaction scoring views.

## Main areas

- `app/`: route-level UI pages and feature components.
- `public/`: static assets.
- `package.json`: scripts and frontend dependencies.

## Run

- `npm install`
- `npm run dev`

## Important notes

- Verdict terminology in UI is `ALLOW / FLAG / BLOCK`.
- The live page consumes backend responses and demo streams for visualization.
