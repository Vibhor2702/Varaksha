# Frontend App Routes

This folder contains Next.js app routes and shared UI code.

## Routes

- `page.tsx`: landing and overview pitch page.
- `live/`: live transaction stream and panels.
- `flow/`: architecture and processing flow walkthrough.
- `timeline/`: project timeline and milestone narrative.
- `lib/`: route-level configuration helpers.

## Purpose

Provide a complete storytelling and operational demo surface for the ML stack.

## Development tip

When backend contract changes, validate `live/` route verdict parsing and counters first.
